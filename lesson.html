Добро пожаловать на первый урок!

Здесь ты напишешь свою первую программу на Python всего за 5–7 минут

👋 Урок 1: Привет, мир! + переменные

1. Установи Python (если ещё не установлен)

Перейди по ссылке и скачай последнюю версию (рекомендую 3.11 или 3.12):

Скачать Python

Во время установки обязательно поставь галочку «Add Python to PATH»

2. Открой терминал / командную строку

  • Windows → Win + R → введи cmd → Enter
  • macOS → Spotlight (Cmd + Space) → Terminal
  • Linux → Ctrl + Alt + T

3. Запусти Python и первую программу

В терминале введи команду: 

python

или (если не сработало)

python3

Когда увидишь >>> — это значит, что Python запущен. Теперь копируй и вставляй по одной строке: 

print("Привет, мир! Я только что запустил Python 🚀")

name = input("Как тебя зовут? ")
print("Круто, " + name + "! Добро пожаловать в мир программирования.")

age = int(input("Сколько тебе лет? "))
if age >= 18:
    print("Ты уже взрослый разработчик!")
else:
    print("Ты молодец, что начал так рано!")

Нажми Enter после каждой строки. Когда закончишь — выйди из Python командой exit() или Ctrl+Z / Ctrl+D.
